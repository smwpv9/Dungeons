File System Information:

Each file with have a 'Sample*.txt' to look at so more can be added to it. One can also use the program to add to the lists.

Most things are databased in the 'Base' folder. This includes Items broken into catagories that represent what they are, and Monsters from the guide book.

The adventures are in their own folder.  These will include the DM file, the character folder, the map folder, and the NPC folder.  There will also be a storyline folder.
